# RegisterEdge

注册边

## 使用说明

> G6 [自定义图项](https://antv.alipay.com/zh-cn/g6/1.x/tutorial/custom-shape.html) 教程

```jsx
import GGEditor, { Flow, RegisterEdge } from 'gg-editor';

<GGEditor>
  <Flow />
  <RegisterEdge name={...} config={...} extend={...} />
</GGEditor>
```

## API

| 属性 | 说明 | 类型 | 默认值 |
| :--- | :--- | :--- | :--- |
| name | 边名称 | `string` | - |
| config | 边配置 | `object` | - |
| extend | 继承图形 | `string` | - |
